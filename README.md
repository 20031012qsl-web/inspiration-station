# AI 训练师 + 业余设计师个人网站

这是一个轻量静态前端项目，用于展示 AI 训练实验、多维视觉作品、3D 建模舞台、经历与个人介绍。

## 本地预览

由于页面使用了 ES Module 与 Three.js CDN，请通过本地 HTTP 服务打开：

```bash
python -m http.server 8000
```

然后访问：`http://localhost:8000`

## 文件结构

- `index.html`：页面结构与内容。
- `styles.css`：视觉系统、响应式、动效样式。
- `app.js`：滚动 reveal、指针聚光、GSAP 滚动过渡、Three.js 舞台。
- `DESIGN.md`：设计规范。
- `.cursor/rules/project-context.mdc`：项目长期上下文。
- `.prompt`：后续构建提示词。

## 设计方向

电影级多维视觉宇宙：深蓝黑、冷雾空间、暖橙能量点、细线网格、作品星群、Bento 作品归档、单个 3D 建模舞台。
